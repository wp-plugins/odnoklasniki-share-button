== Description ==

Плагин позволяет добавлять Share Button от соц. сети Одноклассники.

Есть несколько настроек:

Выводить или нет на главную страницу.

Положение иконки

Размер иконки

== Installation ==

1. Скачать odnoklasniki.zip в /wp-content/plugins/

2. Распаковать

3. Активировать Плагин через меню в WordPress

4. Используйте Настройки->Odnoklasniki Share Button что бы изменить вид кнопки
