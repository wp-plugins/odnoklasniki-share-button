== Description ==
Плагин позволяет добавлять Share Button от соц. сети Одноклассники.
Есть несколько настроек:
=Выводить или нет на гласную страницу.
=Положение иконки
=Размер иконки
=Текст подписи
== Installation ==
1. Upload odnoklasniki.zip to the /wp-content/plugins/ directory
2. Unzip it
3. Activate the plugin through the Plugins menu in WordPress
4. Use the Settings->Odnoklasniki Share Button page to change your plugin options.
